微信小程序轮播图尺寸问题

1.轮播图外层容器swiper,每一个轮播项swiper-item
2.swiper标签存在默认样式
width 100%
height 150px
image 存在默认宽度和高度 320*240
swiper 高度无法实现由内容撑开
3.需先根据原图的宽度和高度 等比例给swiper定宽度和高度
假定：原图宽度和高度为 640 * 200px
swiper 宽度 / swiper 高度 = 原图的宽度 / 原图的高度
则 swiper 高度 = swiper 宽度 * 原图的高度 / 原图的宽度
即 height: 100vw * 200 / 640
4.css样式
swiper{

　　width: 100%;
　　 height: calc(100vw * 200 / 640);
　}
　image{
　　width: 100%;
　}
5.给image标签添加mode属性：mode=”widthFix”(如果现有的图片尺寸不一样，则使用mode=”aspectFill”，取几张图片中的最短的显示，以形成后规范，这样标记点才能准确的在图片下方)
　宽度100%，高度自适应

git相关问题

git push后出现Everything up-to-date报错
输入git status查看发现
modified: .idea/workspace.xml

原因出在./idea这个文件夹是多余的，
默认下是把该文件夹一并上传，所以每次上传在commit的环节时许把./idea的勾去掉

若已经传输到远端，则使用指令
git rm -r -f –cached .idea
将远程的./idea文件夹删掉

关于重装云服务器后不能登录的问题
